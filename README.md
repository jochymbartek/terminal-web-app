# Browser-based Terminal Portal

> Secure, single-sign-on access to Debian & Rocky Linux terminals from any browser.  
> **Tech stack:** Keycloak (OIDC) · NGINX (reverse-proxy + WebSocket) · ttyd · Docker Compose · Azure VNet
